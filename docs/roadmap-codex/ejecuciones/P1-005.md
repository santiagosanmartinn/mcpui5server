# Ejecucion P1-005 - Analizador de rendimiento UI5

Fecha: 2026-03-10
Estado: completada

## Objetivo

Agregar una tool de analisis de rendimiento UI5 sobre XML/JS con hallazgos accionables, reglas estables y severidades consistentes.

## Cambios aplicados

1. Nueva tool MCP:
- `src/tools/ui5/analyzePerformance.js`
- Nombre: `analyze_ui5_performance`
- Capacidades:
  - escaneo de archivos XML y JS
  - reglas de rendimiento con severidad `low|medium|high`
  - salida con `rule`, `message`, `suggestion`, `file`, `line`, `category`
  - resumen agregado por severidad y por regla

2. Registro:
- `src/tools/index.js`

3. Pruebas:
- `test/tools/analyzeUi5Performance.tool.test.js`
- Cobertura:
  - validacion de severidades consistentes
  - validacion de reglas y sugerencias accionables

## Criterios de aceptacion P1-005

- Cada finding incluye regla y sugerencia: OK
- Severidades consistentes: OK

## Evidencia de validacion

- `npm run lint`: OK
- `npm run test:run`: OK
- `npm run check`: OK
- `npm run coverage`: OK

## Notas

- El analizador prioriza hallazgos de alto impacto y evita ruido con reglas acotadas.
- El limite `maxFindings` permite controlar volumen para clientes MCP.
