# Ejecucion P1-006 - Integracion de docs oficiales

Fecha: 2026-03-10
Estado: completada

## Objetivo

Mejorar las tools de documentacion oficial (`search_ui5_sdk`, `search_mdn`) incorporando:
- timeout configurable,
- cache local opcional,
- trazabilidad de fuente y fecha de consulta.

## Cambios aplicados

1. Capa de cache local:
- `src/tools/documentation/cacheStore.js`
- Funciones:
  - normalizacion de opciones de cache
  - lectura/escritura de entradas en `.mcp-cache/documentation`
  - claves deterministas por hash

2. Mejoras en tools:
- `src/tools/documentation/searchUI5SDK.js`
- `src/tools/documentation/searchMDN.js`
- Capacidades nuevas:
  - entrada `timeoutMs`
  - entrada `cache` (`enabled`, `ttlSeconds`, `forceRefresh`)
  - salida `trace` (`queriedAt`, `fetchedAt`, estado cache, timeout)

3. Pruebas:
- `test/tools/documentationSearch.tool.test.js`
- Cobertura:
  - hit/miss de cache
  - trazabilidad de metadata
  - timeout configurable en contrato de salida

## Criterios de aceptacion P1-006

- Trazabilidad de fuente y fecha: OK
