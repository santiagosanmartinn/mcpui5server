# Ejecucion P0-005 - Refactor JavaScript basado en AST

Fecha: 2026-03-10
Estado: completada

## Objetivo

Reemplazar el refactor textual de JavaScript por transformaciones AST para mejorar fiabilidad y trazabilidad de cambios.

## Cambios aplicados

1. Nueva utilidad AST:
- `src/utils/refactor.js`
- Funcion principal:
  - `refactorJavaScriptWithAst(code)`

2. Migracion de tool:
- `src/tools/javascript/refactorCode.js` ahora usa la utilidad AST y mantiene contrato de salida (`refactoredCode`, `changes`).

3. Reglas AST implementadas:
- `var` -> `const` o `let` segun analisis de bindings.
- conversion de handlers Promise (`then`, `catch`) de `function` a `arrow` cuando no usan `this`, `super`, `new.target` ni `arguments`.
- deteccion de callback nesting para recomendacion.
- normalizacion de trailing whitespace.

4. Errores tipificados:
- `INVALID_REFACTOR_INPUT`
- `REFACTOR_PARSE_ERROR`

## Pruebas agregadas

- `test/utils/refactor.test.js`
- Cobertura de casos:
  - conversion `var` + Promise handlers
  - no conversion si hay lexical bindings inseguros
  - conversion en loop initializer
  - deteccion de callback nesting
  - parse error con `ToolError` estable
