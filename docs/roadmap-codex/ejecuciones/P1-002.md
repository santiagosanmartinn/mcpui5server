# Ejecucion P1-002 - Generador UI5 end-to-end

Fecha: 2026-03-10
Estado: completada

## Objetivo

Agregar la tool `generate_ui5_feature` para crear en un solo comando:
- `view` XML,
- `controller`,
- `fragment` (opcional),
- actualizacion de `routing/targets` en `manifest.json`,
- sincronizacion de claves base en `i18n`.

Con soporte de `dryRun` y aplicacion segura via patch.

## Cambios aplicados

1. Nueva tool MCP:
- `src/tools/ui5/generateFeature.js`
- Nombre: `generate_ui5_feature`
- Capacidades:
  - flujo completo de scaffolding UI5
  - `dryRun` por defecto
  - `allowOverwrite` para proteger archivos scaffold existentes
  - preview por archivo (hash + diff)
  - apply real con `applyProjectPatch`
  - validacion pre/post de `manifest`

2. Registro de tool:
- `src/tools/index.js`

3. Pruebas:
- `test/tools/generateUi5Feature.tool.test.js`
- Cobertura funcional:
  - dry-run sin tocar disco
  - apply real e idempotencia
  - rechazo de sobreescritura accidental (`FEATURE_FILE_EXISTS`)

4. Documentacion:
- `README.md`
- `docs/arquitectura-interna.md`
- `docs/referencia-tools.md`
- `docs/uso-servidor.md`
- `docs/ejemplos-tools.md`
