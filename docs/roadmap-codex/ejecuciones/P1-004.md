# Ejecucion P1-004 - Gestor i18n

Fecha: 2026-03-10
Estado: completada

## Objetivo

Incorporar una tool para:
- extraer literales no internacionalizados,
- detectar claves faltantes y no usadas,
- generar reporte por archivo + resumen global,
- ofrecer modo `fix` con `preview/apply`.

## Cambios aplicados

1. Nueva tool MCP:
- `src/tools/ui5/manageI18n.js`
- Nombre: `manage_ui5_i18n`
- Capacidades:
  - `mode=report` para analisis sin cambios
  - `mode=fix` para generar parches sobre XML/i18n
  - `dryRun` por defecto en fix
  - `previews` + `applyResult` con infraestructura segura de patch

2. Registro:
- `src/tools/index.js`

3. Pruebas:
- `test/tools/manageUi5I18n.tool.test.js`
- Cobertura:
  - reporte de faltantes/no usadas
  - fix en dry-run sin modificar disco
  - fix aplicado con patch real
